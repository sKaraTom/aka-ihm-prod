# Akachan

Ma première application.
Middleware java, Front avec Angular.

## En quoi ça consiste ?

Une application d'aide à la recherche de prénom en vu d'un futur heureux événement par exemple.
Elle est faite sous forme de jeu, de suggestions de prénoms qui peuvent être sauvegardés dans sa liste afin d'y concentrer ses recherches, sa réflexion.
